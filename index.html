Driftline API

Driftline detects fraudulent behavior on digital platforms by analyzing messaging metadata in real time. You send us events — we return risk scores. No message content ever leaves your server.

Base URL: https://driftline-api-production.up.railway.app

All endpoints require an X-API-Key header with your API key. Get your key by creating an account.

Quickstart

Get up and running in under 5 minutes. Every time a user sends a message on your platform, call the /event endpoint. We return a risk score immediately.

Python
# Install: pip install requests
import requests

API_KEY = "your_api_key_here"
BASE_URL = "https://driftline-api-production.up.railway.app"

# Call this whenever a user sends a message
# reply_speed = seconds since user's last message in this conversation
# (0 if this is their first message)

response = requests.post(
    f"{BASE_URL}/event",
    headers={"X-API-Key": API_KEY},
    params={
        "user_id": "user_123",
        "conversation_id": "conv_456",
        "platform": "your_platform_name",
        "reply_speed": 2.3,  # seconds since last message
        "message_length": 45   # character count
    }
)

result = response.json()
print(result)
# {
#   "risk_score": 87,
#   "risk_level": "critical",
#   "flags": ["45 msgs/hr", "Bot-like reply speed"],
#   "recommendation": "Suspend account immediately"
# }

# Take action based on risk level
if result["risk_level"] == "critical":
    suspend_user(result["user_id"])

Authentication

All API requests must include your API key in the X-API-Key header.

cURL
curl https://driftline-api-production.up.railway.app/stats \
  -H "X-API-Key: your_api_key_here"

If your API key is missing or invalid, you'll receive a 401 Unauthorized response.

POST /event

The core endpoint. Call this every time a user sends a message on your platform. Returns a real-time risk score.

POST/event

Parameters

ParameterTypeRequiredDescription
user_idstringRequiredYour platform's unique identifier for the user
conversation_idstringRequiredUnique ID for the conversation or chat thread
platformstringRequiredYour platform name (e.g. "bark", "myapp")
reply_speedfloatOptionalSeconds since user's last message in this conversation
message_lengthintegerOptionalCharacter count of the message
timestampfloatOptionalUnix timestamp. Defaults to current time

Response

JSON
{
  "user_id": "user_123",
  "risk_score": 87.0,
  "risk_level": "critical",
  "flags": ["45 msgs/hr", "Bot-like reply speed: avg 1.2s"],
  "flag_reason": "45 msgs/hr",
  "recommendation": "Suspend account immediately",
  "analyzed_at": "2026-03-19T22:00:00"
}

Returns aggregate statistics for your platform.

GET/stats

Returns the current risk score for a specific user.

Removes a user from the flagged list and clears their event history. Use this after reviewing and taking action on an account.

DELETE/account/{user_id}

Python Example

Complete integration example showing how to hook into a messaging system.

Python
import requests
import time

DRIFTLINE_KEY = "your_api_key"
DRIFTLINE_URL = "https://driftline-api-production.up.railway.app"

# Track last message time per conversation
last_message_times = {}

def on_message_sent(user_id, conversation_id, message_text):
    # Calculate reply speed
    now = time.time()
    reply_speed = now - last_message_times.get(conversation_id, now)
    last_message_times[conversation_id] = now

    # Send to Driftline
    result = requests.post(
        f"{DRIFTLINE_URL}/event",
        headers={"X-API-Key": DRIFTLINE_KEY},
        params={
            "user_id": user_id,
            "conversation_id": conversation_id,
            "platform": "your_platform",
            "reply_speed": reply_speed,
            "message_length": len(message_text)
        }
    ).json()

    # Act on result
    if result["risk_level"] == "critical":
        flag_for_review(user_id, result["flags"])
    elif result["risk_level"] == "high":
        add_to_watchlist(user_id)

    return result

JavaScript Example

JavaScript
const DRIFTLINE_KEY = 'your_api_key';
const DRIFTLINE_URL = 'https://driftline-api-production.up.railway.app';

async function checkMessage(userId, conversationId, replySpeed) {
  const params = new URLSearchParams({
    user_id: userId,
    conversation_id: conversationId,
    platform: 'your_platform',
    reply_speed: replySpeed
  });

  const res = await fetch(`${DRIFTLINE_URL}/event?${params}`, {
    method: 'POST',
    headers: { 'X-API-Key': DRIFTLINE_KEY }
  });

  const result = await res.json();

  if (result.risk_level === 'critical') {
    console.log('🚨 Critical risk:', result.flags);
  }

  return result;
}
